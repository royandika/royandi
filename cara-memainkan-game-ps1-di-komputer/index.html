Cara Memainkan Game PS1 Di Komputer

Cara Memainkan Game PS1 Di Komputer

Kawan yang sudah puas dengan asyiknya main PS2 menggunakan flashdisk, mungkin sekali-kali pengen juga bernostalgia dengan game-game PS1 alias PSX atau PSone yang pernah booming di Indonesia sebelum generasi PS2 lahir. Walau terkesan sedikit jadul dibandingkan PS2, game-game di PS1 menurutku secara pribadi masih tetap menghibur untuk dimainin saat ini, karena perbedaan paling mendasar antara game PS1 dan game PS2 (biasanya) adalah masalah kualitas grafik saja, sementara storyline/plot dan keseruan dari game-game PS1 tidak kalah dan jauh berbeda dari game-game PS2. Karena itulah menurutku game-game PS1 masih sangat layak dimainkan saat ini.

Serunya lagi, kini game-game PS1 tidak harus dimainkan di konsol PS1 saja karena berkat tersedianya berbagai emulator, game-game PS1 dapat dimainin di laptop/PC bahkan iPad atau tablet Android. Namun artikel kali ini kita akan bahas cara memainkan game PS1 di komputer atau laptop terlebih dahulu. Adapun untuk cara memainkan game PS1 di komputer, kawan dapat mengikuti cara-cara berikut:

1. Siapkan Emulator PS1

Emulator merupakan software yang akan membuat kita dapat memainkan game PS1 melalui komputer. Tersedia berbagai macam jenis emulator, namun aku menyarankan menggunakan ePSXe yang merupakan salah satu emulator PS1 terbaik karena mudah digunakan, mendukung banyak perangkat, dan ketersediaan fitur save dan cheat yang lengkap.

Kawan dapat men-download ePSXe terbaru di official web mereka dimari. Namun ePSXe yang di download di official web mereka tersebut merupakan versi clean dan kawan memerlukan tambahan plugin dan file BIOS yang mesti di download dari link luar. Nah, biar ga repot, berikut ini kusertakan link download ePSXe 1.9 versi komplit yang bisa langsung kawan jalankan.

2. Konfigurasi ePSXe
Setelah di download dan di ekstrak, jalankan file ‘ePSXe.exe’. Untuk yang mendownload ePSXe 1.90 dari www.epsxe.com, maka akan otomatis masuk ke “Wizard Configuration”. Bagi yang mendownload paket ePSXe + Plugins + BIOS, maka masuk ke menu Config – Wizard Guide.

Klik Config.

Pilih nama file BIOS (sudah otomatis terpilih), klik Next.

Pilih plugin grafis yang sesuai dengan spek komputer, jika kurang paham pilih “psx emulation cheater 2.5”. Lalu klik Config dan Config (lagi) untuk masuk ke menu konfigurasi video.

Di sini kawan dapat melakukan settingan sesuai kebutuhan terutama pengaturan resolusi, warna dan tekstur grafis untuk disesuaikan dengan kemampuan grafik komputer kita. Jika sudah, klik OK dan Next.

Untuk settingan Sound klik Next aja.

Untuk settingan Cdrom juga klik Next aja.

Di menu Configuring Pads ini kita dapat melakukan pengaturan untuk bermain menggunakan controller PC dan support untuk dua stik kontroler (dua pemain). Untuk penyetingan lebih lanjut akan kutuliskan di artikel  bagian bawah. Klik Next.

Klik Done. Konfigurasi ePSXe selesai.

3. Menyiapkan file game PS1
ePSXe dapat memainkan file game PS1 dengan berbagai format antara lain: ISO, IMG, CUE, CCD, MDS, PBP, ECM, ZIP dan EXE. Kawan juga bisa memainkan langsung dari kaset CD game. File-file game PS1 bisa kawan dapatkan melalui dua cara:
Pertama, kawan dapat melakukan ripping terhadap kaset game PS1 menjadi ISO dengan bantuan software seperti USButil, persis seperti yang biasa dilakukan pada kaset game PS2.
Kedua, kawan dapat men-download file-file game PS1 di forum Indowebster (untuk link lokal) atau di www.emuparadise.me (link luar). File game PS1 terhitung kecil, berkisar dari 100MB hingga 700MB, jadi cukup kecil untuk di download dengan kecepatan internet rumahan. Yang perlu diperhatikan adalah format file hasil download apakah didukung oleh ePSXe atau tidak.

4. Menggunakan PC Controller
Game PS1 memang dirancang untuk dimainkan dengan controller, sehingga memainkan game tersebut dengan keyboard tentu lebih sulit. Dengan modal 50ribuan, kawan sudah dapat membeli controller game untuk PC yang bentuk, tampilan dan fungsinya sama dengan controller untuk konsol PS1 dan PS2, sehingga memainkan game PS1 di komputer terasa lebih nyaman.
Cara memainkan game PS1 di komputer menggunakan controller hanya butuh sedikti pengaturan pada ePSXe. Colok controller kawan pada USB komputer, lalu buka ePSXe dan masuk ke menu Config – Game Pad – Port 1 – Pad 1.

Pada dropdown di kanan atas pilih jenis controller kawan, yaitu [SCPH-1150] Dual Analog (F5), lalu cara pengaturannya adalah: Klik dengan mouse pada kotak yang mewakili tombol pada Controller. Misal, klik dengan mouse pada kotak yang kuberi tanda kotak merah (pada gambar, yang mewakili tombol ‘Select’ pada Controller. Setelah di-klik dengan mouse, kini tekan tombol ‘Select’ pada Controller kawan.

Ulangi langkah tersebut pada tombol yang lain, klik dengan mouse pada area kotak yang mewakili tombol Start, lalu tekan tombol Start pada Controller, dan seterusnya hingga semua tombol tersetting.

Terakhir, untuk memainkan game PS1, kawan tinggal klik File – Run ISO atau File – Run PS-EXE dan selamat bernostalgia dengan game-game PS1. Semoga bermanfaat.
