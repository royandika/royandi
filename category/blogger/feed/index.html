Blogger telah menambahkan fitur redirect domain blogspot agar dapat disesuaikan dengan domain nasional pembaca blog. Contohnya begini, misalkan saja alamat blog kawan adalah [nama_blog].blogspot.com, lalu ada netter dari Australia yang membuka alamat blog kawan, maka Blogger secara otomatis akan mengganti alamat blog kawan menjadi [nama_blog].blogspot.com.au. Perubahan ini hanya pada domain saja, tanpa sedikitpun merubah tampilan dan isi blog.

Tujuan dari Country Specific Domain Redirection dari blogspot ini antara lain untuk memudahkan Blogger melakukan pembatasan atau pelarangan konten yang mengandung isu-isu tertentu pada negara tertentu. Misalnya, ada konten dari blog kawan yang dapat menyinggung atau membuat marah masyarakat Belanda (misalnya), maka Blogger hanya perlu memblokir domain [nama_blog].blogspot.nl saja, sehingga blog tersebut tidak dapat diakses di Belanda, namun di negara lain tetap bisa diakses. Ada berapa jenis nama domain yang termasuk dalam fitur redirect domain nasional pada blogsspot ini? Antara lain:
  1. Argentina [blogspot.com.ar]
  2. Australia [blogspot.com.au]
  3. Brazil [blogspot.com.br]
  4. Canada [blogspot.ca]
  5. Czech Republic [blogspot.cz]
  6. France [blogspot.fr]
  7. Germany [blogspot.de]
  8. Hong Kong [blogspot.hk]
  9. India [blogspot.in]
  10. Italy [blogspot.it]
  11. Japan [blogspot.jp]
  12. Mexico [blogspot.mx]
  13. New Zealand [blogspot.co.nz]
  14. Portugal [blogspot.pt]
  15. Romania [blogspot.ro]
  16. Spain [blogspot.com.es]
  17. Sweden [blogspot.se]
  18. UK [blogspot.co.uk]
Untuk kawan bloger yang ingin mengatasi masalah redirect domain nasional pada blogspot sehingga blog kawan tidak bisa di redirect menjadi domain-domain nasional diatas, berikut caranya:

Seperti yang sudah kawan-kawan bloger ketahui, Ctrl+U adalah salah satu hotkey atau tombol pintas untuk dapat melihat source code atau kode html suatu laman page melalui browser. Biasanya tombol ini dipakai untuk ‘mengintip’ struktur kode laman page sehingga dapat melakukan copy-paste (copas) pada artikel atau isi blog.

Sejujurnya, menghindari web atau blog kita dari usaha copas dapat dikatakan hampir mustahil, karena selalu ada cara untuk dapat mengintip jeroan web dan blog kita. Namun begitu, memasang satu-dua pengaman sebagai antisipasi mungkin lebih baik daripada tidak memasang pengaman anti hotkey, sehingga web blog kita tidak bisa di copas oleh orang awam. Berikut ini beberapa cara memasang anti Ctrl U di blogspot melalui salah satu dari dua cara berikut:1. Memasang script JavaScript

Buka Dashboarb Blogger – Template – Edit HTML, lalu masukkan kode berikut sebelum kode</head>
<script src="https://googledrive.com/host/0B_lTgxrsLhSZNVZmOERoLWdIajQ" type="text/javascript"></script>
2. Modifikasi tag BODY
Masih di Dashboarb Blogger – Template – Edit HTML, lalu cari tag <body> dan tambah/ubah tag tersebut sehingga menjadi seperti ini:
<body oncontextmenu='return false;' onkeydown='return false;' onmousedown='return false;'>
Ingat, kawan hanya perlu menerapkan salah satu dari cara diatas, tidak perlu dua-duanya. Semoga dengan cara ini web dan blog kawan akan lebih aman dari para blogger pemalas.

Kawan bloger pastinya sudah cukup mengenal dengan yang namanya laman 404 Page Not Found, yaitu sebuah laman yang akan muncul secara otomatis jika web server tidak menemukan url laman web yang diinginkan oleh pengunjung. Blogspot server juga akan menampilkan laman 404 ini jika pengunjung blog menuju url yang tidak tercatat di server blogger. Namun, kini blogpsot mengijinkan kawan bloger untuk memasang tampilan 404 Page Not Found dengan tampilan yang diinginkan.

Sebagai contohnya, kawan bisa melakukan klik pada url yang keliru di RoyAndika.com, misalnya url http://www.royandika.com/12/12/404 dan lihatlah tampilan page 404 yang dimiliki RoyAndika.com. Sudah berbeda dengan tampilan default dari blogspot, kan? Di laman 404 RoyAndika.com, terdapat teks informasi dan disertai dengan redirect countdown, yaitu hitungan mundur yang akan membawa pengunjung blog ke laman utama RoyAndika.com dalam 5 detik. Tentunya laman 404 seperti ini akan lebih bermanfaat untuk membimbing ‘pengunjung yang tersesat’ dan tetap berada di blog kita tanpa banyak usaha.

Contoh Tampilan 404 Page Not Found Dengan Redirect Otomatis Pada RoyAndika.com

Untuk memasang tampilan 404 page not found dengan redirect otomatis pada blogspot ternyata sangat mudah, langkah-langkahnya:

Masuk ke Dashboard blog dan masuk ke menu Setting – Search Preferences

Klik Edit pada Custom Page Not Found dan akan muncul sebuah textbox.

Isi dengan kode berikut:

<script type="text/javascript">
window.onload=function() {
function countdown() {
if ( typeof countdown.counter == 'undefined' ) {
    countdown.counter = 5; // jumlah detik hitungan mundur
    }
if(countdown.counter > 0) {
 document.getElementById('count').innerHTML = countdown.counter--;
    setTimeout(countdown, 1000);
    }
else {
    location.href = 'http://www.royandika.com'; // alamat url yang ingin kawan arahkan
    }
}
countdown();
};
</script>
<div style="text-align: center;">
<b>404</b><br />
PAGE NOT FOUND!
<br /><br />
Laman yang kawan cari/tuju sudah tidak tersedia lagi di [nama blog]. Mungkin kawan dapat menemukan laman yang diinginkan melalui index artikel berikut ini!<br /><br />
Will be redirected in <span id="count"></span> seconds...
</div>
Kawan bisa lakukan editing pada kode HTML diatas agar tampilannya lebih menarik. Setelah itu, klik tombol Save.

Untuk melihat hasilnya, kawan bisa ke alamat yang tidak dimiliki blog, misalnya http://[url_blogspot]/2014/03/alamat-error untuk memunculkan laman 404.
