Pada catatanku sebelumnya, untuk memainkan game PS2 menggunakan flashdisk atau HDD menggunakan file ISO game PS2 ternyata menemui berbagai tantangan. Di antara tantangan tersebut adalah tidak semua judul game PS2 yang bisa kita ripping menjadi file ISO lalu dimainkan dengan OpenPS2Loader, seperti yang sudah kucatat di catatanku yang berjudul Cara Bikin ISO DVD Game PS2 Terproteksi. Judul game seperti Dynasty Warriors 6 atau Final Fantasy XII – International Zodiac Job System merupakan judul-judul game yang tidak bisa di-ripping begitu saja menjadi file ISO, karena telah terproteksi.

Pada catatanku berjudul Cara Bikin ISO DVD Game PS2 Terproteksi, saya sudah mencatat tentang cara ripping DVD menjadi file ISO plus cara patching IOPRP, dan dalam catatan kali ini kita khusus mencatat sedikit lebih jauh tentang lika-liku patching IOPRP.

Berdasarkan fitur pada aplikasi USButil, ternyata terdapat 2 (dua) jenis patching untuk mengatasi proteksi pada kaset DVD game PS2, yaitu patching DNAS dan IOPRP. Sampai detik ini pun aku masih tidak tahu perbedaan antara patching DNAS dan IOPRP ini, dan kita hanya akan fokus pada satu jenis patch saja yaitu: IOPRP.

Hampir Semua Judul Game Sudah Ter-Patch IOPRP
Ini adalah kenyataan sederhana yang aku temui sejauh ini, bahwa semua kaset game PS2 yang kucoba ripping menjadi ISO ternyata sudah memiliki patch IOPRP. Cara mengenali file patching IOPRP diantara file-file game pada data kaset DVD cukup mudah, yaitu kawan tinggal mencari file dengan format nama IOPRP***.IMG (tanda bintang merupakan angka). File patch IOPRP ini biasanya dapat ditemukan pada root file data kaset DVD game PS2 seperti berikut:

Atau dapat ditemukan dalam folder IOP seperti berikut:

Jadi pada dasarnya hampir semua judul game PS2 yang dijual di pasaran sudah memiliki file patch IOPRP, sehingga bisa dimainkan pada konsol PS2 kawan. Aku juga menyimpulkan, entah bagaimana, sepertinya patch IOPRP ini memiliki keterkaitan dengan modchip yang terpasang pada konsol, mungkin bahkan keduanya berpasangan (misalnya modchip fungsingya memang mengaktifkan patch IOPRP-nya). Tapi tentu saja itu hanya dugaan, karena aku sama sekali bukan ahli patching game, hehehe…

Tidak Semua Patch IOPRP Pada Kaset DVD Berjalan Juga Pada ISO
Walau kebanyakan kaset game PS2 yang dijual di pasaran sudah memiliki patch IOPRP secara default, namun ternyata tidak menjamin patch tersebut akan jalan pada saat game tersebut berbentuk file ISO. Disinilah fungsi patching IOPRP pada fitur aplikasi USBUtil diperlukan, yaitu kawan dapat mem-patch ulang game PS2 dalam format ISO file yang tidak dapat dimainkan dengan OpenPS2Loader. Hal ini terjadi pada beberapa judul game seperti Dynasty Warriors 6 dan God Hand.

Patching IOPRP Pada Format ISO dan GAME
Kawan yang sudah membaca catatanku yang berjudul Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD, terdapat dua jenis instalasi game pada flashdisk atau HDD yaitu dengan format ISO dan format GAME. Untuk format ISO, kawan dapat melakukan patching IOPRP secara langsung pada file ISO, baik pada file ISO yang kawan mainkan secara aktif (tersimpan di flashdisk) maupun pada file ISO yang kawan back-up ke HDD.

Namun berbeda dengan format GAME dimana proses patching sebaiknya dilakukan dengan cara:
1. Menggunakan source file ISO yang masih ‘fresh’ (belum di patch dengan USButil), atau source langsung langsung dari kaset DVD game PS2.
2. Patching IOPRP dilakukan pada saat melakukan format GAME ke dalam FD/HDD menggunakan USButil.
3. Jika kawan menggunakan source file ISO untuk diformat sebagai GAME ke FD/HDD, maka sebaiknya kawan melakukan backup pada file ISO tersebut, karena proses format file ISO ke GAME ke dalam FD/HDD, USButil akan mem-patch file ISO terlebih dahulu secara otomatis lalu melakukan format menjadi GAME dan di transfer ke FD/HDD. Hal ini membuat file ISO kawan ter-patch dan termodifikasi sehingga beresiko merusak file ISO tersebut.

File ISO yang sudah ter-patch tidak dapat lagi dibuka dengan aplikasi compress/decompress file seperti 7zip, sehingga menyulitkan kita melakukan pengolahan lebih jauh terhadap file ISO tersebut, seperti misalnya untuk memperkecil ukuran file ISO tersebut, yang mungkin akan kucatat dalam catatan tersendiri nanti. Semoga bermanfaat.

Dalam catatanku sebelumnya yang berjudul Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD telah kutuliskan perbedaan format instalasi game PS2 di flashdisk dengan format ISO dan GAME, lengkap dengan tutorial cara instalasi game PS2 ke flashdisk dengan masing-masing format. Namun sejatinya memang tidak semua judul game PS2 yang bisa di bikin ISO lalu mulus lancar jaya dimainin dengan OpenPS2Loader (OPL). Ada beberapa judul game yang sudah dibuat file ISO-nya lalu dibuka dengan OPL hanya akan menampilkan layar kosong berwarna saja, tanpa ada tampilan opening game menu seperti seharusnya.

Dalam kondisi seperti ini, yang bisa kawan lakukan pertama kali adalah mengecek:

  1. Apakah ukuran file ISO game tersebut lebih besar dari 4 GB? Jika memang lebih dari 4GB, maka sebaiknya mencoba instalasi game dengan format GAME seperti yang kutuliskan pada artikel Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD.
  2. Jika ukuran file ISO game tersebut lebih kecil atau sama dengan 4GB, maka kemungkinannya adalah DVD game PS2 tersebut terproteksi.

Proteksi pada kaset DVD game ini merupakan jenis proteksi yang akan melindungi kaset DVD dari usaha pembajakan, seperti penggandakan kaset dan peng-copy-an file data DVD. Salah satu kasus yang kutemukan adalah pada kaset DVD game Final Fantasy XII International Zodiac Job System yang ternyata file ISO-nya gagal dieksekusi oleh OPL. Untuk mengatasi hal ini, cara bikin ISO DVD game PS2 yang terproteksi tersebut adalah dengan memanfaatkan fasilitas patch yang sudah disediakan oleh aplikasi USButils pada saat melakukan ripping DVD game PS2 menjadi file ISO.

Cara bikin ISO DVD game PS2 yang terproteksi adalah sebagai berikut:
Pertama-tama, gunakan aplikasi USButil v2.00 – update DNAS Pack v2.1 yang bisa di download di sini. Lalu jalankan program tersebut hingga muncullah tampilan berikut:

Pilih File → Create [GAME/ISO] from CD/DVD

Lalu sesuaikan settingan berikut:

Patch to [GAME/ISO] → Centang IOPRP
Extract as → pilih ISO
Destination → tentukan lokasi hasil file ISO
Name to GAME/ISO → beri nama file ISO, misalnya: final_fantasy_xii.iso
Klik Create

Proses ripping DVD game PS2 berjalan, jika sudah selesai (100%) klik Close dan file ISO sudah siap digunakan.

Sebenarnya yang perlu diperhatikan adalah jenis patch yang kita pilih, apakah DNAS atau IOPRP. Sejujurnya saya tidak tahu perbedaan antara patching DNAS atau IOPRP, namun pada kasus game Final Fantasy XII International Zodiac Job System saya sudah melakukan ripping ke ISO dengan kombinasi berikut:

  • Ripping ISO tanpa patcing DNAS/IOPRP → OPL gagal eksekusi
  • Ripping ISO dengan patching DNAS/IOPRP (kedua-duanya dicentang) → OPL gagal eksekusi
  • Ripping ISO dengan patcing DNAS saja (tanpa patching IOPRP) → OPL gagal eksekusi
  • Ripping ISO dengan patching IOPRP saja (tanpa patching DNAS) → OPL berhasil eksekusi

Jadi, walaupun perbedaan jenis patching DNAS/IOPRP masih menjadi misteri bagiku, namun dengan beberapa kali mencoba kombinasi diatas akhirnya aku berhasil bikin ISO DVD game PS2 yang terproteksi dan bisa dimainin dengan lancar. Jika suatu saat kawan mendapati judul game yang gagal dieksekusi oleh OPL, maka ada baiknya mencoba satu persatu kombinasi patching tersebut. Semoga bermanfaat.

Ga nyangka juga ternyata catatan-ku yang berjudul Main PS2 Menggunakan Flashdisk mendapat respon yang cukup bagus bahkan menjadi salah satu catatan favorit. Dan karena cukup banyak permintaan untuk menuliskan langkah-langkah instalasi agar bisa main PS2 menggunakan flashdisk, maka kali ini aku akan catatkan tutorial main PS2 menggunakan flashdisk.

Sebenernya catatan tutorial main PS2 menggunakan flashdisk ini penulisan ulang dari tutorial di blog ini, yang aku coba tulis ulang dengan lebih rapi dan mudah dimengerti. Tutorial ini berhasil diterapkan pada konsol PS2 Slim SCPH-77000 milikku dan mudah-mudahan bisa diterapkan oleh kawan-kawan yang memiliki konsol yang sama.

Bahan-bahan yang perlu disiapkan antara lain:

  1. Flashdisk kapasitas minimal 4GB, lebih besar lebih baik.
  2. Kaset CD blank + CD/DVD burner untuk mem-burn aplikasi ke CD
  3. Paket aplikasi untuk instalasi yang bisa di download antara lain: uLE v4.38 boot CD ISO atau uLE v4.38 boot DVD ISO, ELF_PSU.rar
  4. Memory Card PS2 (MC) yang sudah dikosongkan terlebih dahulu

Pertama, format flashdisk melalui PC dengan format FAT32.

Ekstrak ELF_PSU.rar dan pindahkan semua file dan foldernya ke dalam flashdisk.

Kedua, ekstrak ‘uLE v4.38 boot CD ISO.zip‘ atau ‘uLE v4.38 boot DVD ISO.zip‘ (pilih salah satu) untuk mendapatkan file ISO. Burn file ISO tersebut ke kaset CD atau DVD blank yang sudah kawan sediakan, disesuaikan jenis ISO nya CD atau DVD.

Ketiga, mainkan kaset CD/DCD yang sudah di-burn dengan aplikasi uLE v.38 boot dengan konsol PS2. Sekalian juga flashdisknya dan MC-nya dicolokkan ke konsol PS2. Ada kemungkinan kaset CD/DVD tidak bisa langsung terbaca oleh konsol, jika begitu coba ulangi beberapa kali hingga terbaca atau ganti dengan kaset CD/DVD baru. Jika sudah terbaca maka akan muncul aplikasi uLaunch seperti ini:
Tekan tombol [O] untuk masuk ke FileBrowser. Tombol [O] berfungsi sebagai tombol Enter, sedangkan tombol [X] sebagai tombol Cancel atau Back, dan tombol [R1] sebagai tombol menu.
Pada FileBrowser terdiri dari beberapa folder dengan penjelasan:
mc0:/ → folder memory card slot 1
mc1:/ → folder memory card slot 2
hdd0:/ → folder harddisk eksternal
mass:/ → folder flashdisk
Keempat, masuk ke mass:/ lalu pada file BOOT.psu tekan [R1] dan pilih Copy. Kembali ke menu awal FileBrowser dengan tombol [△] dan masuk ke mc0:/ lalu klik [R1] dan pilih PsuPASTE. Kemudian masuk ke mass:/ dan COPY semua file *.ELF lalu PASTE ke mc0:/BOOT (buat dulu folder BOOT dalam mc0:/).
Kelima, masuk ke folder mass:/FreeMcBoot lalu tekan lingkaran [O] pada file FREE MCBOOT.ELF dan kawan akan di bawa ke MC Boot instaler.
Silahkan lakukan ‘normal install’ (untuk MC yang dimainkan hanya pada satu konsol PS2) atau ‘multi version install’ (untuk MC yang dimainkan pada banyak konsol PS2 yang berbeda). Tunggu hingga sampai proses finish.
Matikan konsol PS2, copot kaset CD/DVD-nya dan hidupkan kembali konsol PS2 dengan MC tetap tercolok. Jika MC sudah berhasil terinstall FreeMCBoot, maka akan muncul tambahan menu seperti gambar berikut:
Pilih Free McBoot Configurator lalu tekan [X] jika ada kotak keterangan kecil yang muncul. Kemudian pilih Configure OSDSYS options, lalu pilih configure item 1 : ulaunchelf. Tekan [X] lalu sesuaikan dengan settingan berikut:

item 1 settings
name: uLaunchELF → ganti nama sesuai keinginan
path1: mass:/BOOT/BOOT.ELF → MC0:/BOOT/BOOT.ELF
path2: MC?:/BOOT/BOOT.ELF → hapus/clear
path3: MC?:/B?DATA-SYSTEM/BOOT.ELF → hapus/clear

Jika sudah disesuaikan, tekan [△] kemudian pilih configure item 1. Tekan tombol kanan stik pada bagian configure item 2 : ESR, lalu tekan [X] atau OK.

Item 2 settings
Name: ESR → Ganti dengan nama OpenPS2Loader
path1: MASS:/BOOT/ESR.ELF → MC0:/BOOT/OPNPS2LD.ELF
path2: MC?:/BOOT/ESR.ELF → hapus/clear
path3: MC?:/B?DATA-SYSTEM/ESR.ELF → hapus/clear

Jika sudah selesai disesuaikan, tekan [△] sampai ke menu awal pada free mcBOOT configurator kemudian pilih save CNF to MC0. Reset/restart konsol PS2.

Setelah konsol PS2 restart lalu masuk ke OpenPS2Loader (memory card tetap terpasang). Setelah loading nanti akan muncul halaman dari OPL-nya (berwarna biru background-nya). Pilih settings lalu ganti baris berikut:

USB device start mode off ubah menjadi auto

  1. File ISO terdiri dari 1 buah file, biasanya 1 kaset DVD game akan diubah oleh USButils menjadi sebuah file ISO.

  3. Game dengan format file ISO sangat mudah diinstall.

Namun sebelum memindahkan, flashdisk mesti disiapkan terlebih dahulu. Lalu pindahkan file ISO tadi langsung ke dalam folder ‘CD’ jika besaran file tidak lebih dari 1 GB, atau ke folder ‘DVD’ jika besaran file lebih besar dari 1GB.   1. Format GAME terdiri dari beberapa file yang merupakan pecahan atau part yang terbagi berdasarkan dengan ukuran besaran file.
  2. Format GAME digunakan untuk game-game dengan besaran file yang lebih besar dari 4GB, sehingga akan dipecah menjadi beberapa part dengan ukuran file yang relatif sama (biasanya setiap part masing-masing memiliki besaran file ±1.00 GB, kecuali file part terakhir yang lebih kecil).
  3. Instalasi dengan format GAME hanya bisa melalui aplikasi USButils atau USB Extreme. Sumber game untuk di-convert ke format GAME bisa langsung dari kaset DVD maupun file ISO (file ISO bisa di convert ke format GAME).

    Contoh penyimpanan game dengan format ISO dan GAME secara bersamaan

  6. Jika kawan melakukan ripping langsung dari kaset DVD menjadi format GAME, kawan juga bisa mengkonversi format GAME tersebut menjadi format ISO melalui USButils, sehingga bermanfaat sebagai backup.

Ternyata ini dikarenakan sebelumnya aku sudah me-replace file BOOT.ELF default (saat instalasi booting dari MC) dengan file BOOT.ELF dari file Codebreaker hasil download yang mana file tersebut ternyata mengubah secara otomatis booting MC yang tadinya ke banyak menu start up, sekarang otomatis membuka OpenPS2Loader! WTF! Bagaimana dong solusinya? Solusi tersederhana yang kupakai adalah dengan menjalankan Ulaunch yang sudah di burn ke CD (bekas instalasi booting PS2 dari MC), dan dengan begini masalahnya terselesaikan. Jadi simpelnya begini: kawan jalankan CD Ulaunch, lalu masuk ke folder MC0 dan jalankan Codebreaker.ELF (jangan lupa cabut FD/HDD terlebih dahulu), pilih cheat dan ubah option Auto Tray Eject menjadi OFF, lalu START GAME, sebelum tekan OK pasang kembali FD atau HDD, tekan X untuk OK, dan OpenPS2Loader akan jalan secara otomatis. Sekarang kita sudah bisa main game PS2 di flashdisk atau harddisk dengan cheat! Horeee….
