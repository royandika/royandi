Terhitung sejak kemarin, tanggal 8 Juli 2014, pengelolaan komentar blog www.RoyAndika.com telah migrasi menggunakan layanan moderasi komentar dari Disqus. Alasan mengapa aku memilih menggunakan Disqus adalah selain memiliki tampilan yang lebih ciamik, namun juga karena kemampuan dan fitur yang mendukung banyak platform serta sosial media sehingga sangat memudahkan kawan-kawan meninggalkan ‘jejak’.

Namun Disqus, secara default, tidak menyediakan fitur untuk menampilkan komentar terbaru sebagai widget Blogspot. Disqus juga tidak menyediakan link RSS/Feed untuk komentar-komentar pada website, sehingga sedikit menyulitkan untuk memasang widget recent post Disqus pada Blogger. Namun, untungnya ada sebuah cara mudah untuk memasang widget recent post Disqus pada Blogger seperti yang aku tampilkan pada widget kanan bawah blogku ini. Cukup masukkan kode berikut di widget blogspot kawan: 
<div id="recentcomments" class="dsq-widget">
<script type="text/javascript" src="http://nama_blog.disqus.com/recent_comments_widget.js?num_items=5&hide_avatars=1&avatar_size=50&excerpt_length=10">
</script>
</div>

Jangan lupa untuk mengganti ‘nama_blog’ pada kode tersebut dengan nama blog kawan sesuai dengan alamat dashboard Disqus blog kawan.

Blogger memiliki fitur bar navigasi yang muncul disetiap blog dari blogger. Bar navigasi ini, atau sering disebut juga dengan Navbar Blogger, bisa sangat bermanfaat bagi bloger maupun pembaca, namun juga seringkali membuat tampilan blog menjadi terganggu terutama pada blog yang menggunakan desain template sendiri (bukan dari tampilan yang disediakan Blogger).
Beberapa fitur dari Navbar Blogger adalah:

  1. Ikon B jika diklik akan menuju ke blogger.com
  2. Link pintas ke Dashbor Blogger dengan jendela “Sign In / Sign Out”
  3. Kolom pencarian google
  4. Dapat memberikan raitng pada blog melalui fitu “Flag Blog”
  5. Dengan fitur NextBlog dapat menuju ke blog lain secara acak

Kerugiannya hanyalah pada tampilan. Jika kawan menggunakan tampilan dengan desain sendiri, Navbar blogger seringkali merusak keseluruhan tampilan. Bagaimana cara menghilangkan navbar blogger?

Cara 1: Dari Widget
Masuk ke Dashboard – Layout
Pada deretan widget klik Edit pada widget bernama Navbar
Klik Off dan Save

Cara 2: Dari Template Design (cara lama)
Masuk ke Dashboard – Template – Edit Template
Cari bagian CSS, biasanya diantara tag berikut: 

<style type='text/css'>
Kode CSS
</style>

Masukkan tag berikut: 

#navbar { height: 0px; visibility: hidden; display: none;}
#navbar-iframe { height: 0px; visibility: hidden; display: none;}

Klik Save
Sekarang kawan sudah berhasil menghilangkan Navbar blogger dari tampilan blog.
