]]> Base URL pada CodeIgniter adalah untuk menentukan nama alamat (url) homepage aplikasi CodeIgniter. Untuk menyetingnya kita harus membuka file config.php yang ada di direktori /system/application/config lalu kita dapat mengedit bagian ini: 
$config['base_url'] = "http://example.com/";

Biasanya kita dapat menyetingnya menjadi url domain (jika hosting online) seperti http://www.royandi.com, atau jika kita masih localhost maka menggunakan http://localhost/namafolder. Ada satu alternatif yang (mungkin) bisa digunakan sebagai alternatif untuk localhost yaitu menggunakan alamat:

$config['base_url'] = "http://".$_SERVER["SERVER_NAME"]."/namafolder";

Kegunaannya adalah kita menggunakan alamat ‘umum’ dari server lokal jadi apapun engine Apache-nya, tetap bisa jalan!
