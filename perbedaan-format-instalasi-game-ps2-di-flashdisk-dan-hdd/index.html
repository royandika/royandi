Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD

Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD

Bagi kawan yang sudah bisa memainkan game PS2 menggunakan flashdisk atau HDD, serta bisa menginstall game PS2 ke flashdisk atau HDD secara mandiri, tentu sudah mengetahui bahwa dalam melakukan instalasi game ke flashdisk atau HDD agar dapat dimainkan terdapat 2 (dua) cara yaitu dengan format ISO dan format GAME. Pada artikel kali ini akan kucoba cari tahu sebenarnya apa sih perbedaan antara format ISO dan GAME ini?

Sebelum kita mengetahui perbedaan format instalasi game PS2 di flashdisk dengan format ISO dan GAME, ada baiknya kita mengetahui kesamaannya terlebih dahulu, antara lain:

  1. Pertama, kedua format ini harus disimpan dalam flashdisk dengan format FAT32 (format default flashdisk) karena konsol PS2 hanya dapat membaca format storage ini.
  2. Kedua, baik format ISO maupun format GAME dapat dimainkan dengan OpenPS2Loader.
  3. Ketiga, file kedua format ini juga dapat di ripping secara langsung (dan tak langsung) dari kaset game DVD PS2 menggunakan software USButils.

Sekarang kita akan bahas perbedaannya satu per satu:

Game Dengan Format ISO

  1. File ISO terdiri dari 1 buah file, biasanya 1 kaset DVD game akan diubah oleh USButils menjadi sebuah file ISO.
  2. Format ISO sebaiknya digunakan untuk game-geme yang besaran file-nya lebih kecil dari 4GB. Kenapa? Karena sepengalamanku beberapa game yang ketika menjadi file ISO memiliki besaran lebih dari 4GB ternyata tidak dapat di copy-paste ke flashdisk. Hal ini mungkin dipengaruhi oleh format flashdisk FAT32 yang memang didesain untuk flashdisk dengan kapasitas kecil.
  3. Game dengan format file ISO sangat mudah diinstall.
  4. Cara install: Setelah kawan melakukan ripping game DVD PS2 menggunakan software USButils, maka kawan akan mendapatkan sebuah file ISO yang siap dipindahkan ke flashdisk. Namun sebelum memindahkan, flashdisk mesti disiapkan terlebih dahulu. Pastikan flashdisk dalam dormat FAT32 dan buatlah dua buah folder baru dengan nama masing-masing ‘CD’ dan ‘DVD’ (tanpa tanda kutip yah). Lalu pindahkan file ISO tadi langsung ke dalam folder ‘CD’ jika besaran file tidak lebih dari 1 GB, atau ke folder ‘DVD’ jika besaran file lebih besar dari 1GB. Setelah itu copy-kan juga software iso2OPL.exe ke folder utama flashdisk (folder terluar tempat folder ‘CD’ dan ‘DVD’ berada). Jalankan software iso2OPL.exe tersebut dan file ISO tadi secara otomatis akan di-rename sesuai dengan nama rilis dari game tersebut (misal game Shadow of Colossus akan di rename menjadi SCUS_974.72.Shadow Of The Colossus). Selesai. Colok flashdisk ke konsol PS2 dan OpenPS2Loader akan dapat mendeteksi file ISO tersebut sebagai file game.

    Contoh menu USButil untuk memformat ISO dan GAME

  5. File ISO sangat mudah disimpan sebagai back-up di HDD atau laptop, karena tidak perlu di-convert ke format lain, tinggal copy-paste ke HDD atau PC/laptop. Dan file ISO yang sudah di rename dengan nama rilisnya (mis. SCUS_974.72.Shadow Of The Colossus) maka ketika kawan copy-paste kembali ke dalam flashdisk, tidak perlu lagi menjalankan aplikasi iso2OPL.exe.
  6. Untuk me-ripping kaset game PS2 menjadi ISO tidak melulu harus dengan menggunakan aplikasi USButils. Kita bisa menggunakan aplikasi ISO image lainnya seperti Nero, ISO Magic atau ImgBurn. Namun yang perlu diperhatikan adalah untuk melakukan setting read speed pada kecepatan 4x (maks. 8x).

Game Dengan Format GAME

  1. Format GAME terdiri dari beberapa file yang merupakan pecahan atau part yang terbagi berdasarkan dengan ukuran besaran file.
  2. Format GAME digunakan untuk game-game dengan besaran file yang lebih besar dari 4GB, sehingga akan dipecah menjadi beberapa part dengan ukuran file yang relatif sama (biasanya setiap part masing-masing memiliki besaran file ±1.00 GB, kecuali file part terakhir yang lebih kecil).
  3. Instalasi dengan format GAME hanya bisa melalui aplikasi USButils atau USB Extreme. Sumber game untuk di-convert ke format GAME bisa langsung dari kaset DVD maupun file ISO (file ISO bisa di convert ke format GAME).
  4. Cara install: Sumber game yang bisa kawan gunakan adalah kaset DVD atau file ISO yang bisa langsung di convert ke format GAME oleh aplikasi USButils. File format GAME ini akan disimpan di folder utama flashdisk. OpenPS2Loader akan mendeteksi keberadaan game dengan format ini melalui sebuah file yang bernama ul.cfg dimana file ul.cfg ini ter-generate secara otomatis saat kita melakukan instalasi/konversi format GAME ke flashdisk melalui aplikasi USButils. Namun (sepertinya) file ul.cfg ini akan ter-replace setiap kali kita melakukan instalasi game baru sehingga game lama (kadang) tidak lagi terbaca oleh OpenPS2Loader. Untuk itu yang perlu kita lakukan hanyalah melakukan update file ul.cfg melalui USButils dengan pilihan menu Utils → Recover list of games.

    Contoh menu Utils dari USButil

  5. Kawan bisa menyimpan game dengan format GAME dan ISO secara bersamaan, karena disimpan di folder berbeda. Format GAME disimpan di folder utama flashdisk, sedangkan format ISO disimpan di folder ‘CD’ atau ‘DVD’.

    Contoh penyimpanan game dengan format ISO dan GAME secara bersamaan

  6. Jika kawan melakukan ripping langsung dari kaset DVD menjadi format GAME, kawan juga bisa mengkonversi format GAME tersebut menjadi format ISO melalui USButils, sehingga bermanfaat sebagai backup.
Begitulah perbedaan format instalasi game PS2 di flashdisk dan HDD, semoga bisa membantu kawan mempermudah instalasi game PS2 pada flashdisk.
