Cara Bikin File ISO Dari Kaset DVD Game PS2 Yang Terproteksi

Cara Bikin File ISO Dari Kaset DVD Game PS2 Yang Terproteksi

Dalam catatanku sebelumnya yang berjudul Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD telah kutuliskan perbedaan format instalasi game PS2 di flashdisk dengan format ISO dan GAME, lengkap dengan tutorial cara instalasi game PS2 ke flashdisk dengan masing-masing format. Namun sejatinya memang tidak semua judul game PS2 yang bisa di bikin ISO lalu mulus lancar jaya dimainin dengan OpenPS2Loader (OPL). Ada beberapa judul game yang sudah dibuat file ISO-nya lalu dibuka dengan OPL hanya akan menampilkan layar kosong berwarna saja, tanpa ada tampilan opening game menu seperti seharusnya.

Dalam kondisi seperti ini, yang bisa kawan lakukan pertama kali adalah mengecek:

  1. Apakah ukuran file ISO game tersebut lebih besar dari 4 GB? Jika memang lebih dari 4GB, maka sebaiknya mencoba instalasi game dengan format GAME seperti yang kutuliskan pada artikel Perbedaan Format Instalasi Game PS2 Di Flashdisk Dan HDD.
  2. Jika ukuran file ISO game tersebut lebih kecil atau sama dengan 4GB, maka kemungkinannya adalah DVD game PS2 tersebut terproteksi.

Proteksi pada kaset DVD game ini merupakan jenis proteksi yang akan melindungi kaset DVD dari usaha pembajakan, seperti penggandakan kaset dan peng-copy-an file data DVD. Salah satu kasus yang kutemukan adalah pada kaset DVD game Final Fantasy XII International Zodiac Job System yang ternyata file ISO-nya gagal dieksekusi oleh OPL. Untuk mengatasi hal ini, cara bikin ISO DVD game PS2 yang terproteksi tersebut adalah dengan memanfaatkan fasilitas patch yang sudah disediakan oleh aplikasi USButils pada saat melakukan ripping DVD game PS2 menjadi file ISO.

Cara bikin ISO DVD game PS2 yang terproteksi adalah sebagai berikut:
Pertama-tama, gunakan aplikasi USButil v2.00 – update DNAS Pack v2.1 yang bisa di download di sini. Lalu jalankan program tersebut hingga muncullah tampilan berikut:

Pilih File → Create [GAME/ISO] from CD/DVD

Lalu sesuaikan settingan berikut:

Patch to [GAME/ISO] → Centang IOPRP
Extract as → pilih ISO
Destination → tentukan lokasi hasil file ISO
Name to GAME/ISO → beri nama file ISO, misalnya: final_fantasy_xii.iso
Klik Create

Proses ripping DVD game PS2 berjalan, jika sudah selesai (100%) klik Close dan file ISO sudah siap digunakan.

Sebenarnya yang perlu diperhatikan adalah jenis patch yang kita pilih, apakah DNAS atau IOPRP. Sejujurnya saya tidak tahu perbedaan antara patching DNAS atau IOPRP, namun pada kasus game Final Fantasy XII International Zodiac Job System saya sudah melakukan ripping ke ISO dengan kombinasi berikut:

  • Ripping ISO tanpa patcing DNAS/IOPRP → OPL gagal eksekusi
  • Ripping ISO dengan patching DNAS/IOPRP (kedua-duanya dicentang) → OPL gagal eksekusi
  • Ripping ISO dengan patcing DNAS saja (tanpa patching IOPRP) → OPL gagal eksekusi
  • Ripping ISO dengan patching IOPRP saja (tanpa patching DNAS) → OPL berhasil eksekusi

Jadi, walaupun perbedaan jenis patching DNAS/IOPRP masih menjadi misteri bagiku, namun dengan beberapa kali mencoba kombinasi diatas akhirnya aku berhasil bikin ISO DVD game PS2 yang terproteksi dan bisa dimainin dengan lancar. Jika suatu saat kawan mendapati judul game yang gagal dieksekusi oleh OPL, maka ada baiknya mencoba satu persatu kombinasi patching tersebut. Semoga bermanfaat.
