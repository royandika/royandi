Untuk file ISO yg disimpan sejahu yg sy tau tidak akan ada kerusakan apapun kecuali karena:
(1) Secara otomatis oleh software tertentu (misalnya emulator atau USButils);
(2) Tidak sengaja oleh kita sendiri
Saya mmg pernah mengalaminya, dan antisipasi yg sy pakai adalah:
(1) Jika memainkan game dengan emulator, pastikan untuk menggunakan file ISO hasil copy dari file ISO aslinya, hal ini bisa menghindari terjadinya editing oleh emulator.
(2) Coba cara ini: klik kanan pada file ISO, pilih properties dan centang pada Read Only. Cara ini mungki bisa menghalangi 'usaha' emulator untuk melakukan editing tanpa sepengetahuan kita…
