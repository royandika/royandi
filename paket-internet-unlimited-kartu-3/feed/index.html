Matikan dulu mobile networksnya (hilangkan centang 'Use packet data' pada Menu – Settings – Wireless and networks – Mobile networks.

Lalu buka Menu – Settings – Wireless and networks – Mobile networks – Access Point Names
Pilih APN yg ingin di edit, atau jika mau aman tekan tombol menu dan pilih 'New APN'. Lalu masukkan parameter APN kartu 3 berikut:
Name: 3 data (nb: bebas kasih nama apa saja)
APN: 3data
Proxy: Not set (nb: biarkan kosong)
Port: 80
User name: 3data
Password: 3data
Server: Not set (nb: biarkan kosong)
MMSC: Not set (nb: biarkan kosong)
MMS Proxy: Not set (nb: biarkan kosong)
MMS Port: Not set (nb: biarkan kosong)
MCC: 510 (nb: nilai bawaan, jangan diubah)
MNC: 89 (nb: nilai bawaan, jangan diubah)
Authentication type: PAP
APN type: internet

Setelah oke semua, klik 'Menu' dan 'Save'. Lalu pada layar APNs pastikan sudah terpilih APN yg baru kita buat tadi.

Aktifkan kembali Mobile Networks dengan mencentang 'Use packet data'. Silahkan dicoba, semoga berhasil.
