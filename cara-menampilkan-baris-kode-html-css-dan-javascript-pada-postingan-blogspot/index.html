Cara Menampilkan Baris Kode HTML, CSS Dan JavaScript Pada Postingan Blogspot

Kawan-kawan sesama bloger yang memiliki blog dengan tema seputar tips atau tutorial IT, seringkali dalam postingan harus memasukkan baris-baris kode pemrograman sebagai bahan referensi bagi pembaca blog. Seperti tutorial pemrograman web yang banyak menggunakan baris-baris kode HTML, PHP, atau JavaScript yang sering menjadi bahan untuk di-share kepada sesama bloger.

Untuk menuliskan baris-baris kode-kode tersebut dalam postingan blogspot dengan tampilan yang rapi dan cantik, RoyAndika.com akan membagikan salah satu cara menampilkan baris kode HTML, PHP dan JavaScript pada postingan Blogspot. Dan tidak hanya untuk bahasa HTML, CSS atau JavaScript aja, kawan juga bisa menuliskan baris bahasa PHP, SQL, dan CSS. Dan semua tampilan baris kode yang terdapat dalam postingan ini atau postingan lain di RoyAndika.com merupakan hasil dari cara ini.

Adapun keuntungan yang kawan-kawan peroleh dari cara ini antara lain:

  1. Lebih rapi
  2. Lebih menarik
  3. Mudah dibaca
  4. Terdapat nomor baris yang memudahkan pencarian
  5. Mudah dipasang pada blogspot
  6. Ringan dan tidak membebani blog

Untuk memasangnya pada blogspot, pertama kawan harus melakukan pengeditan pada template design blogspot. Caranya masuk ke Dashboard – Template.
Sebelumnya lakukan backup template agar bila kawan gagal atau terjadi kesalahan pada saat editing template, dapat mengembalikan tampilan blog seperti kondisi awal. Klik tombol Back Up / Restore (di kanan atas), dan silahkan download template dalam format XML.
