1. Versi PAL dan NTSC itu versi apanya bro? Region file game kah atau option pada saat mainnya? Sejauh ini saya tidak pernah 'terlibat' dengan versi ini (pada saat proses instalasi game)
2. Emulator yg digunakan emulator apa dan berjalan di mana? Apakah di konsol PS2 atau di PC?

Proses yang saya sarankan ringkasnya adalah:

a. Ambil kaset DVD game-nya, masukkan ke PC. Teliti size-nya, apakah lebih atau kurang dari 4 GB. Klu kurang dari 4 GB, kita akan format sebagai ISO. Klu lebih dari 4 GB kita format GAME. (baca: royandika.com/2012/10/perbedaan-format-instalasi-game-ps2-di.html).

b. Gunakan USBUtil v2.0 untuk mem-build data dari kaset DVD ke Flashdisk (baik itu format ISO maupun GAME)

c. Coba mainkan dengan OpenPS2Loader. Akan ada splash screen warna warni sebelum game dijalankan.

d. Kalau layarnya mentok di salah satu warna (gagal menjalankan file), maka masbro boleh coba defrag file ISO/GAME nya dengan aplikasi Auslogics Defragment dan jalankan fitur defrag single file. Setelah selesai, coba lagi jalanin lagi dengan OpenPS2Loader.

e. Jika masih gagal (mentok pada splash screen warna-warni), ada kemungkinan kaset DVD game tersebut terproteksi, maka diperlukan patching pada saat building ke FD melalui USBUtil. Baca lagi catatan diatas (Lika-Liku Patching ISO Game PS2). Prinsip dari patching ini adalah, masbro coba build game di FD dengan mencoba satu persatu: tanpa patching apapun, patching IOPRP saja, patching DNAS saja, dan patching IOPRP+DNAS. Coba satu persatu, dan defrag tiap kali selesai di build.

f. Sampai tahap ini jika masih gagal juga, kemungkinan besar game tsb termasuk salah satu game yg tidak bisa dimainkan dengan OpenPS2Loader (ingat tidak semua game bisa dimainkan).

g. Terakhir, dari keterangan masbro 'bahkan game yang semulanya lancar juga ikut2an gagal' maka dapat dipastikan instalasi OpenPS2Loader di memory card sudah corrupt (rusak), jadi masbro mesti format dan install ulang MMC nya, dan coba lagi. Karena kasus mirip pernah sy alami ternyata masalahnya mmg pd instalasi pd MMC yg corrupt, bukan (lagi) pd file game di FD.
