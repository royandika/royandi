Cara Root Samsung Galaxy Tab 3 Dengan Odin

Cara Root Samsung Galaxy Tab 3 Dengan Odin

Berbeda dengan catatanku sebelumnya yang berjudul Cara Root Samsung Galaxy Tab 3 Paling Cepat Dan Sederhana, catatanku kali ini akan merunutkan tata cara root Samsung Galaxy Tab 3 (SM-T211) melalui Odin, khususnya untuk perangkat yang sudah update firmware ke versi build number JZO54K.T211XXANB2 (tanggal update rilis 18 Februari 2014). Hal ini berkenaan dengan catatanku kemarin yang berjudul A Nightmare Called Bootloop yang mana perangkat dengan update firmware terbaru belum support di-root dengan Kingo dan dapat mengakibatkan bootloop.

Selain itu, cara root Samsung Galaxy Tab 3 (SM-T211) dengan Odin ini sudah kucoba sendiri dan berjalan dengan lancar. Prinsip langkahnya sebenarnya sangat sederhana yaitu kawan cuma perlu menginstall sebuah file Zip yang berada di SD Card melalui mode Recovery. Hanya saja permasalahannya firmware official tidak menyediakan fitur install file Zip pada mode Recovery-nya, jadi kawan harus melakukan instalasi CWM (Clockworkmod) untuk memodifikasi mode Recovery bawaan Samsung.

PERHATIAN!
Melakukan rooting pada perangkat Android akan mengakibatkan kehilangan garansi resmi. Rooting beresiko mengakibatkan perangkat mati total sementara maupun selamanya, dan RoyAndika.com tidak bertanggungjawab atas segala hasil yang diakibatkan oleh perlakuan rooting pada perangkat. DO WITH YOUR OWN RISK!

Sebelum memulai, yang perlu kawan persiapkan antara lain:

  1. ODIN v3.07
    Yaitu aplikasi yang menjadi ‘jembatan’ antara komputer PC dan perangkat Tab 3. Terdapat versi yang lebih baru, namun belum kucoba kompatibilitasnya dengan Tab 3, silahkan cari dan coba sendiri versi updated-nya jika berminat.
  2. T211.CWM.v6.0.4.5.recovery.tar.zip
    Merupakan aplikasi Clockwordmod untuk modifikasi recovery mode Tab 3.
  3. Universal Root
    Paket update berformat ZIP yang akan me-root perangkat.
  4. Pastikan baterai perangkat diatas 50%, jika perlu charge dulu hingga penuh.
  5. Pastikan perangkat dapat terkoneksi dengan baik ke komputer melalui kabel USB. Jika kawan dapat melakukan transfer file dari perangkat ke PC (atau bisa terkoneksi dengan KIES), berarti tidak ada masalah. Namun jika belum pernah terkoneksi dengan PC, maka kawan perlu meng-install driver Samsung di PC dengan driver yang bisa di download di sini.

Cara Instalasi Clockwordmod (CWM) pada Samsung Galaxy Tab 3 7″ (SM-T211)
Ekstrak ODIN v3.307

Ekstrak juga file T211.CWM.v6.0.4.5.recovery.tar.zip untuk mendapatkan file recovery.tar.md5

Matikan perangkat Tab 3 hingga benar-benar mati. Masuk ke mode download dengan menekan dan menahan selama 5 detik tiga tombol berikut secara bersamaan: Power, Volume Bawah (-) dan Home, lalu tekan tombol Volume Atas (+) pada layar tampilan peringatan. Jika berhasil masuk ke mode download, maka akan muncul tampilan berikut pada perangkat:

Colokkan kabel USB ke perangkat dan ke PC agar terkoneksi. Buka folder ODIN, klik kanan pada file ‘Odin3 v3.07.exe’ dan pilih Run as Administrator. Tampilan ODIN akan seperti ini:

Pada panel ID:COM (ditandai kotak merah pada kiri atas), merupakan penanda perangkat telah terhubung dengan ODIN. Sedangkan panel Message (kotak merah kiri bawah) merupakan laporan status antara ODIN dan perangkat.

Perhatikan hal berikut: Beri centang pada Auto Reboot dan F. Reset Time, dan jangan sekali-kali memberi centang pada Re-Partition, karena beresiko membuat perangkat mati total (Hard Brick).

Klik tombol PDA dan browse untuk mencari file ‘recovery.tar.md5’ dan klik OK. Pastikan hanya tombol PDA yang dicentang. Klik START untuk memulai instalasi CWM pada perangkat. Tunggu sebentar hingga perangkat melakukan reboot dan dapat menyala secara normal, dan jangan lupa untuk melepas koneksi kabel USB dan menutup ODIN. Setelah perangkat dapat beroperasi secara normal, maka kita bisa lanjutkan ke proses Rooting.

Cara Root pada Samsung Galaxy Tab 3 7″ (SM-T211)
Pindahkan file Universal Root yang berformat ZIP ke SD Card perangkat Tab 3, jangan di ekstrak dan biarkan tetap dalam format ZIP. Lalu matikan perangkat sampai benar-benar mati.

Masuk ke Recovery Mode dengan cara menekan dan menahan selama 5 detik tiga tombol berikut secara bersamaan: Power, Volume Atas (+) dan Home. Tampilan Recovery Mode akan seperti berikut:

Tampilan Recovery Mode official, jika sudah terinstall CWM akan terdapat pilihan ‘Install Zip’

Geser menggunakan tombol volume dan pilih menggunakan tombol power. Pilih ‘Install ZIP’ lalu pilih Choose Zip from sdcard. Cari file Universal Root yang sudah kawan simpan di SD Card lalu tunggu proses hingga selesai. Setelah selesai, kembali ke menu sebelumnya dan pilih Reboot system now.

Setelah perangkat kembali menyala dan beroperasi seperti biasa, kawan dapat melihat terdapat satu program baru terinstall bernama SuperSU yang menandakan Samsung Galaxy Tab 3 sudah dalam keadaan rooted. Untuk memastikannya kawan dapat menginstall aplikasi Root Checker dari Google Play.
