Cara Root Galaxy Tab 3 8.0 SM-T311 Kitkat 4.4.2

Perangkat Samsung Galaxy Tab 3 8.0 (SM-T311) merupakan perangkat yang beruntung karena termasuk dalam daftar penerima update Android Kitkat 4.4 secara resmi dari Samsung. Update Kitkat pada perangkat ini sudah dapat di download secara luas oleh pemilik perangkat sejak September 2014 melalui OTA (Over the Air), Aplikasi KIES, maupun flashing manual via ODIN.

Bagi kawan pemilik perangkat Samsung Galaxy Tab 3 8.0 (SM-T311) yang ingin melakukan root pada perangkat, berikut ini catatan saya tentang Cara Root Samsung Galaxy Tab 3 8.0 (SM-T311) Kitkat 4.4.2 yang cukup mudah dilakukan dan berhasil kuterapkan pada perangkatku sendiri.

Sebelum melakukan root, perhatikan:

  1. Proses root pada dasarnya tidak mengakibatkan kehilangan data seperti kontak, gambar, video, dll. Namun mengingat proses ini beresiko, maka sebaiknya data perangkat di backup terlebih dahulu.
  2. Proses root pada dasarnya aman jika dilakukan dengan benar, namun jika terjadi kesalahan dapat menyebabkan perangkat mengalami kerusakan sistem baik itu bersifat ringan (soft brick) maupun berat (hard brick).
  3. Proses root menyebabkan hilangnya garansi resmi perangkat. Proses unroot tidak menjamin garansi akan kembali berlaku.
  4. DO WITH YOUR OWN RISK. Cara ini berhasil dengan baik pada perangkatku, namun tidak berarti pasti berhasil juga pada perangkat kawan! Jangan menyalahkan siapa-siapa jika ternyata gagal root dan mengakibatkan kerusakan sistem.

Root Samsung Galaxy Tab 3 8.0 (SM-T311) Kitkat 4.4.2 via ODIN

Langkah 1: Download dan install Samsung USB Driver pada komputer kawan. Jika Samsung USB Driver sudah terinstall pada komputer, lewati langkah ini. Samsung USB Driver bisa kawan download di sini.

Langkah 2: Download dan ekstrak file untuk root Samsung Galaxy Tab 3 8.0 (SM-T311) yang bisa kawan download di sini.

Langkah 3: Matikan perangkat tablet (Power Off)

Langkah 4: Nyalakan kembali perangkat dengan cara menekan secara bersamaan tombol Home, Power dan Volume Down hingga perangkat masuk ke mode Download seperti ini:

Jika perangkat justru menyala seperti biasa dan tidak muncul layar seperti diatas, berarti kawan gagal masuk ke Download Mode dan matikan lagi perangkat lalu coba menekan ketiga tombol 5-8 detik. Jika sudah masuk ke Download Mode, tekan tombol Volume Up untuk melanjutkan dan muncul layar berikut:


Langkah 5: Buka folder hasil ekstrak pada langkah 2, dan buka Odin3. Lalu koneksikan perangkat ke komputer dengan kabel charger.

Langkah 6: Odin akan mendeteksi perangkat secara otomatis ditandai dengan pesan Added pada bagian bawah. Jika Odin belum mendeteksi perangkat, silahkan kawan coba untuk koneksi ulang perangkat hingga berhasil.

Langkah 7: Klik tombol PDA dan pilih file CF-Auto-Root.tar

Langkah 8 : Klik tombol Start

Langkah 9 : Proses ini biasanya memakan waktu kurang lebih 1 -5 menit dan selama proses rooting, perangkat akan mengalami reboot secara otomatis. Proses root Samsung Galaxy Tab 3 8.0 SM-T311 Kitkat 4.4.2 akan selesai ketika muncul pesan PASS! di kotak hijau. Diskoneksikan perangkat dari komputer.

Langkah 10 : Sekarang perangkat sudah dalam keadaan root, dan untuk mengeceknya bisa dilihat dari daftar aplikasi di Drawer yaitu adanya penambahan aplikasi SuperSu. Selamat mencoba.
