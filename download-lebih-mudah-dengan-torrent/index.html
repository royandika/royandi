Download Lebih Mudah Dengan Torrent

Apa itu torrent?
Torrent adalah metode download data/file antar komputer dengan model P2P (peer to peer). Tidak seperti metode download pada umumnya dimana komputer kita (sebagai client) mendownload dari server host (seperti Indowebster, Mediafire, 4shared, dll), dengan torrent kita dapat mendownload data/file yang di share langsung dari komputer orang lain (seeder). Setiap komputer yang berbagi data/file tersebut bertindak sebagai peer (download) sekaligus sebagai seeder (upload).

Kelebihan torrent

  • 100% gratis, baik aplikasi downloader-nya maupun file yang di download
  • 99,99% PASTI bisa di pause dan resume
  • Server hostnya banyak (tergantung jumlah seed), sehingga ketika terputus dari satu host digantikan oleh host-host lain.
  • Tidak ada fitur Premium yang membatasi kecepatan download maupun fitur resumeable
  • Tidak rakus bandwidth
  • Tidak membuat lelet aktifitas browsing
  • Sebagian besar file (biasanya video atau musik) sudah bisa dibuka walau belum selesai didownload
  • Sangat jarang menyebabkan file hasil download broken atau corrupted
  • Tidak ada pemecahan sebuah file menjadi beberapa part
  • Dapat mendonload banyak file dengan cara di-antri (queue) ataupun sekaligus

Kekurangan torrent

  • Harus cermat dalam memilih file torrent
  • Kecepatan download bervariasi, tergantung dari ketersediaan seed (host) dan peer (client)
  • Lebih rentan terhadap virus dan malware
  • Ketersediaan dan umur file torrent tidak selalu terjamin, file torrent yang tidak populer seringkali tidak bisa di download lagi setelah beberapa waktu

Cara mendownload dengan torrent

  • Download aplikasi Torrent, bisa BitTorrent atau uTorrent
  • Download file torrent yang mewakili link untuk mendownload file
  • Buka file Torrent dengan aplikasi Torrent dan mulailah men-download
  • Aturlah kecepatan upload dan download agar sesuai dengan kebutuhan dan ketersediaan bandwidth
  • Aturlah direktori penyimpanan agar tidak memboroskan kapasitas harddisk
  • Aturlah queue dan priority agar dapat mendahulukan file tertentu
  • Setelah selesai 100%, COPY hasil download dari direktori dan jangan lekas meng-cut atau menghapus file tersebut dari direktori hasil download torrent (setidaknya untuk 2-3 hari, lebih lama lebih baik)
  • Jadilah Peer dan Seeder yang baik dan turut berpatisipasi, jangan menjadi leecher yang egois dan tidak tahu diri

  • Perbandingan jumlah seed (upload) dan peer/leecher (download). Jaminan download cepat adalah seed > peer/leecher atau setidaknya memiliki rasio 1:1
  • Rating dan komentar para user terhadap file hasil download
  • SELALU AKTIFKAN DAN UPDATE ANTIVIRUS
  • Jangan burru-buru menghapus torrent yang sudah 100%, berpartisipasilah sebagai Seeder dengan membiarkan file tersebut terupload selama beberapa hari

Istilah yang perlu diketahui
Torrent file
Adalah file yang mengandung link download yang akan dibaca dan diatur oleh aplikasi Torrent downloader
Seed atau seeder
Adalah istilah untuk jumlah komputer user yang menyediakan file download (sebagai host)
Peer
Adalah istilah untuk jumlah komputer user yang sedang men-download file
Leecher
Adalah istilah untuk jumlah komputer user yang hanya berniat men-download tanpa berniat menjadi seeder.
uTorrent atau BitTorrent
Adalah nama aplikasi Torrent manager yang dapat membaca file torrent dan mengatur proses download dan upload.
thepiratebay.se atau kickass.to
Adalah website penyedia file torrent terbesar yang paling sering digunakan
