Download, Extract dan Convert PS2 Game Saves Dari Gamefaqs

Download, Extract dan Convert PS2 Game Saves Dari Gamefaqs

Kawan yang membaca catatanku ini kuasumsikan sudah membaca terlebih dahulu catatan-catatanku sebelumnya yaitu Download File Save Untuk Memory Card Game PS2 dan PS2 Game Saves – Bermain Curang Tanpa Kecurangan. Bukan apa-apa sih, maksudnya biar nyambung gitu dengan kawan-kawan yang membaca sehingga aku ga perlu mengulangi lagi penjelasan-penjelasan seputar PS2 Game Saves.

Selain di blog ini, kawan juga bisa men-download file game saves dari website Gamefaqs, dimana kawan bisa menemukan file game saves dari berbagai judul game dan dengan beberapa pilihan versi game saves yang bisa kawan download sesuai dengan kecocokan dan kebutuhan game yang kawan mainkan. Namun sayangnya file game saves hasil download dari Gamefaqs tersebut tidak praktis karena tidak bisa langsung kawan copy-paste ke memory card begitu saja. Hal ini dikarenakan hasil download dari Gamefaqs berupa 1 buah file dengan ekstensi yang beragam, bisa .zip, .max, .cbs, atau .xps. Ekstensi-ekstensi tersebut mengindikasikan jenis aplikasi yang  digunakan player untuk mem-backup game saves dari MMC PS2.

Agar dapat di copy-paste ke MMC kawan, pertama-tama kawan harus meng-extract terlebih dahulu file  hasil download tersebut, dengan mengikuti tutorial singkat berikut:

Aplikasi yang kawan butuhkan adalah PS2 Save Builder yang bisa di download di sini. Ekstrak dan akan tersedia file-file berikut:

Jalankan file ps2save-builder.exe sehingga muncul tampilan berikut:

Klik File – Open dan pilih file game saves hasil download. Dalam contoh kali ini kugunakan file game saves Final Fantasy XII International Zodiac Job System yang di download dari link Gamefaqs. File tersebut bernama ‘final_fantasy_xii_intl_jp.cbs’. Setelah itu akan muncul tampilan berikut:

Pastikan Root/ID game saves sudah sesuai dengan ID pada file ISO game. Lalu klik semua file yang muncul dalam daftar (ctrl+klik kiri).

Klik kanan dan pilih Extract, lalu save pada lokasi yang diinginkan. Untuk file name biarkan bernama ‘not used’.

Maka akan ter-ekstrak file-file game saves seperti pada gambar berikut:

Terakhir, simpan semua file tersebut dalam folder yang diberi nama dengan nama ID game tersebut. Nama ID game tersebut bisa dilihat pada nama salah satu file, dalam contoh diatas adalah file yang bernama BISLPM-66750FF1200, dan kawan harus memberi nama folder dengan nama yang sama persis, yaitu BISLPM-66750FF1200.

Dengan begitu kawan sudah berhasil men-download, extract dan convert PS2 game saves dari Gamefaqs untuk di copy-paste ke MMC secara langsung. Untuk cara memindahkan file game saves hasil ekstraksi tadi ke memory card game PS2 bisa dilihat pada catatanku di sini. Semoga bermanfaat!
