Lika Liku Instalasi Codebreaker ELF Pada PS2 Slim

Lika Liku Instalasi Codebreaker ELF Pada PS2 Slim

Setelah sebelumnya aku sudah berhasil main PS2 menggunakan flashdisk atau harddisk tanpa optik, seperti yang sebelumnya sudah kucatat di sini, kini langkah selanjutnya adalah bagaimana cara memainkan PS2 menggunakan flashdisk/harddisk dengan cheat. Jika kawan masih memainkan PS2 dengan menggunakan optik, maka tidak perlu repot lagi karena sudah tersedia berbagai macam alat bantu cheat, salah satunya seperti Action Replay MAX yang sudah dalam bentuk kaset DVD.

Namun tentu tidak semudah itu untuk yang memainkan PS2 menggunakan flashdisk atau harddisk, karena Action Replay MAX dalam bentuk DVD setelah kita mengaktifkan cheat yang akan digunakan, maka secara otomatis akan mem’boot’ game dari kaset menggunakan optik, sementara yang kita inginkan tentu saja setelah mengaktifkan cheat maka game akan di ‘boot’ dari file ISO game di flashdisk atau harddisk. Dan sayangnya Action Replay MAX tidak memiliki option untuk mengubah ‘booting’ game dari kaset DVD ke file ISO.

Solusinya adalah dengan menggunakan Codebreaker, yaitu aplikasi yang mirip dengan Action Replay MAX, yaitu untuk memainkan game PS2 dengan cheat. Codebreaker memiliki option untuk mengubah booting game dari CD/DVD Tray ke file dalam folder. Uniknya, Codebreaker tersedia dalam bentuk:

  1. CD
    Untuk Codebreaker dalam bentuk kaset CD bisa kawan dapatkan dengan men-download Codebreaker ISO CD di sini. Setelah di download, silahkan di burn ke kaset CD dengan kecepatan paling rendah!
  2. ELF
    Dengan Codebreaker dalam bentuk file ELF, kita dapat menginstall Codebreaker langsung pada Memory Card (MC) PS2. Kawan dapat men-download Codebreaker ELF di sini, dan cara install-nya dan cara menggunakan Codebreaker bisa di baca di kaskus.

Prinsip dasarnya cukup sederhana. Codebreaker ELF cukup diletakkan dalam MC PS2, lalu jalankan Ulaunch dan pilih file Codebreaker ELF untuk dijalankan. Tapi ingat, file BOOT.ELF dalam folder BOOT di MC harus juga di-replace dengan file BOOT.ELF yang kawan download tadi, sehingga Codebreaker akan melakukan booting dan membuka aplikasi OpenPS2Loader setelah kita selesai memilih cheat (sebelumnya option Auto Tray Eject pada Codebreaker harus sudah di OFF kan, baca lagi di Kaskus).

Tapi ketika saya coba install Codebreaker ELF pada PS2 Slim-ku dengan cara tersebut, ternyata Codebreaker ELF yang dieksekusi dari Ulaunch tadi tidak memunculkan apa-apa, hanya layar gelap saja. Apa yang terjadi? Dimana kesalahannya? Apakah versi Codebreaker ELF nya tidak support dengan modchip PS Slim-ku? Setelah guling-guling di gugel, ternyata jawabannya sangat sederhana:

Ketika mengeksekusi Codebreaker ELF di MC dengan Ulaunch, pastikan flashdisk atau HDD kawan tidak terpasang ke PS2, karena akan mengakibatkan pembacaan PS2 menjadi berat! Hasilnya layar menjadi hitam!

Akhirnya kucoba menjalankan Codebreaker dari MC dengan terlebih dahulu mencabut flashdisk atau HDD dari konsol PS2, hidupkan PS2 dengan Tray terbuka (booting dari MC) dan jreng jrengggg…. Terbukalah OpenPS2Loader!!! Harusnya yang terbuka adalah Free MC Boot menu dengan pilihan Browser, System Configuration, Ulaunch (terinstall di MC), OpenPS2Loader (terinstall di MC), dll. Kenapa secara otomatis langsung membuka aplikasi OpenPS2Loader? Ini tentu saja masalah besar karena untuk mengjalankan Codebreaker ELF haruslah membuka Ulaunch terlebih dahulu, baru menjalankan OpenPS2Loader?!

Ternyata ini dikarenakan sebelumnya aku sudah me-replace file BOOT.ELF default (saat instalasi booting dari MC) dengan file BOOT.ELF dari file Codebreaker hasil download yang mana file tersebut ternyata mengubah secara otomatis booting MC yang tadinya ke banyak menu start up, sekarang otomatis membuka OpenPS2Loader! WTF! Bagaimana dong solusinya? Solusi tersederhana yang kupakai adalah dengan menjalankan Ulaunch yang sudah di burn ke CD (bekas instalasi booting PS2 dari MC), dan dengan begini masalahnya terselesaikan. Jadi simpelnya begini: kawan jalankan CD Ulaunch, lalu masuk ke folder MC0 dan jalankan Codebreaker.ELF (jangan lupa cabut FD/HDD terlebih dahulu), pilih cheat dan ubah option Auto Tray Eject menjadi OFF, lalu START GAME, sebelum tekan OK pasang kembali FD atau HDD, tekan X untuk OK, dan OpenPS2Loader akan jalan secara otomatis. Sekarang kita sudah bisa main game PS2 di flashdisk atau harddisk dengan cheat! Horeee….
