Cara Install Aplikasi Pajak e-SPT Masa PPH Pasal 22

Bagi perusahaan teretentu yang melakukan kegiatan usaha ekspor dan impor, maka sesuai ketentuan mengenai pajak untuk memenuhi kewajiban wajib Pajak Penghasilan Pasal 22 (PPh pasal 22). Untuk melakukan input dan pelaporan pajak PPh pasal 22 ini, Dirjen Pajak telah menyediakan aplikasi yang dapat digunakan oleh badan usaha untuk melakukan pelaporan pajak melalui aplikasi e-SPT Masa PPh Pasal 22 versi 2.2.

Berikut ini adalah cara instalasi aplikasi pajak penghasilan e-SPT Masa PPh Pasal 22 di windows.
1. Download aplikasi: Official Link (pajak.go.id) | Google Drive (Mirror)

2. Ekstrak dan jalankan file 22setup.exe untuk memulai instalasi.
3. Klik Next dan Next hingga selesai instalasi

Cara koneksi ke database

Aplikasi pajak Aplikasi Pajak e-SPT Masa PPH Pasal 22 menggunakan file .mdb (Microsoft Access) sebagai database-nya. Lokasi penyimpanan file database ini adalah di C:Program FilesDJPSPT PPh 22Database. Untuk mengkoneksikan antara aplikasi dan database, maka langkahnya:
4. Buka file C:Windowssystem32odbcad32.exe untuk menjalankan menu ODBC Data Sources

5. Tekan tombol Add dan pilih Microsoft Access Driver (*.mdb), klik Finish

6. Pada menu ODBC Microsoft Access Setup, isikan Data Source Name lalu klik Select… dan browse file .mdb di lokasi database tersimpan di C:Program FilesDJPSPT PPh 22DatabasedbF113202.mdb, lalu klik OK hingga semua jendela menu ODBC tertutup.

Instalasi Aplikasi Pajak e-SPT Masa PPH Pasal 22 di Windows dan mengkoneksikan dengan database sudah selesai. Namun sering kali ditemui beberapa kendala berikut ini:

Mengatasi permasalahan format tanggal pada system komputer tidak sesuai

Artinya kita hanya perlu melakukan penyesuaian format tanggal pada komputer agar sesuai dengan format tanggal pada Aplikasi Pajak e-SPT Masa PPH Pasal 22.

Klik OK dan pada menu Region pilih format: Indonesian (Indonesia) dan sesuaikan foramt tanggal sesuai gambar berikut:

Klik Apply dan OK.

Muncul pesan ‘Unable to create DSN’ pada Aplikasi Pajak e-SPT Masa PPH Pasal 22

Hal ini dikarenakan Aplikasi Pajak e-SPT Masa PPH Pasal 22 memerlukan hak akses sebagai Administrator untuk mengkoneksikan dan mengelola database. Untuk mengatasi error ‘Unable to create DSN’ pada Aplikasi Pajak e-SPT Masa PPH Pasal 22 adalah:

1. Buka lokasi C:Program FilesDJPeSPT PPh 22
2. Klik kanan pada file ‘prjeSPTPPhMasa22.exe’ dan pilih Properties
3. Pada tab Compatibility beri centang pada ‘Run this program as administrator’
4. Klik OK

Buka kembali Aplikasi Pajak e-SPT Masa PPH Pasal 22, jika semua sudah sesuai maka akan tampil seperti berikut:

Pilih Data Source Name (DSN) yang telah kita buat sebelumnya pada step 4

Memindahkan Aplikasi Pajak e-SPT Masa PPH Pasal 22 ke Komputer Baru

Untuk memindahkan Aplikasi Pajak e-SPT Masa PPH Pasal 22 di komputer lama ke komputer baru, caranya sangat mudah. Kita hanya perlu memindahkan (copy dan paste) file dbF113202.mdb (lihat step 6) dari komputer lama ke komputer baru yang sudah terinstall Aplikasi Pajak e-SPT Masa PPH Pasal 22. Setelah selesai di copy-paste, ulangi step 4–5–6 untuk mengkoneksikan ulang Aplikasi Pajak e-SPT Masa PPH Pasal 22 dengan file dbF113202.mdb yang baru dipindahkan.

Demikianlah cara Install Aplikasi Pajak e-SPT Masa PPH Pasal 22 Windows 10, semoga dapat bermanfaat.

]]> Melakukan update pada windows melalui fitur bawaan Windows Update seringkali menghadapi kendala, diantaranya mentok pada ikon loading, terputus akibat koneksi tidak stabil, dan sebagainya. Hal tersebut sering terjadi dikarenakan men-download windows update memerlukan quota besar dan kecepatan internet yang stabil.
Untuk mengatasinya, kita dapat melakukan update pada Windows secara manual melalui aplikasi pihak ketiga, dan meng-update secara offline. Maksudnya secara offline adalah, kita dapat men-download terlebih dahulu update-update apa saja yang diinginkan, setelah itu dapat di-install secara terpisah. Aplikasi pertama yang saya bicarakan adalah:

WSUS OFFLINE UPDATE

Kelebihan dari aplikasi wsus offline update ini adalah:

  1. Memilih update untuk windows versi server, Windows 7, Windows 8.1, dan Windows 10
  2. Tersedia untuk arsitektur x86 maupun x64 bit
  3. Tersedia update untuk Ms. Office
  4. Source file di-download langsung dari situs windows/microsoft, dijamin keamanan dan orisinil file-nya
  5. Terpisah untuk fitur download dan instalasi
  6. Aplikasi bersifat portable, dapat disimpan dan dijalankan di flashdisk atau HDD eksternal
  7. Bersifat gratis dan bebas dipakai

Cara melakukan download windows update (secara online) adalah:

  1. Download WSUS offline update disini
  2. Ekstrak atau unzip dan jalankan file UpdateGenerator.exe
  3. Pilih versi Windows dan sesuaikan option dengan keperluan
  4. Tekan tombol Start
  5. Proses download akan berjalan pada Command Prompt

Setelah selesai download windows update, untuk melakukan install update dapat dilakukan secara offline:

  1. Buka folder ‘client’
  2. Jalankan ‘UpdateInstaller.exe’
  3. Pilih update yang diinginkan pada tab ‘Updating’
  4. Tekan tombol Start untuk memulai instalasi
  5. Proses instalasi akan berjalan pada Command Prompt

Silahkan diperiksa status Windows Update pada Settings – Update and Security – Windows Update.

Semoga bermanfaat.

Semakin meningkatnya popularitas Raspberry Pi ternyata membuatku tertarik dengannya. Memangnya apa itu Raspberry Pi? Kawan bisa googling sendiri untuk mencari tau tentang Raspberry Pi karena disini saya akan fokus menulis tentang tips dan informasi cara instalasi dan menggunakan Raspberry Pi. Bagi kawan yang tertarik untuk menjajal mini-computer Raspberry Pi ini, pastinya penasaran dengan harga Raspberry Pi (atau biasa disingkat Raspi) dan apa-apa saja yang dibutuhkan untuk memulainya? Yuk kita intip apa-apa saja Perangkat Yang Dibutuhkan Untuk Memulai Menggunakan Raspberry Pi beserta keterangan dan harga.
1. Board Raspberry Pi

Perangkat ini adalah “bintang utama”-nya dan bertindak sebagai mainboard pada komputer, namun berbentuk mini. Board Raspi ‘hanya’ memiliki prosessor, RAM, slot-slot (USB, LAN, power input dan slot penting lainnya), built-in bluetooth dan wifi (untuk versi 3), serta tidak memiliki storage. Raspi terdiri dari beberapa versi, dan yang paling umum dijual di pasaran adalah Raspberry Pi 2 dan Raspberry Pi 3. Perbedaan keduanya adalah Raspi 2, sebagai versi lama hanya memiliki RAM 512 MB dan belum dilengkapi dengan Wifi (sehingga perlu menambah sendiri) sedangkan Raspi 3 sudah memiliki RAM 1 GB lengkap dengan bluetooth dan Wifi yang sudah built-in. Harga Raspi 3 sendiri adalah $35 atau sekitar Rp. 600.000an di pasaran Indonesia, sementara harga Raspi 2 bisa lebih murah yaitu sekitar Rp. 450.000 s/d Rp. 550.000-an.

2. Wifi Dongle (Antena Wifi USB) untuk Raspi 2

Diperlukan bagi board Raspi 2 yang belum memiliki Wifi built-in. Harga mulai dari Rp. 75.000,-an Casing Board Untuk melindungi board dari sentuhan, benturan, dan lain-lain. Harga bervariasi tergantung bentuk dan bahan, berkisar dari Rp. 70.000,-an hingga Rp. 150.000,-an, yang lebih mahal lagi juga ada.

3. Power Adapter

Sebagai sumber power, Raspi bisa menggunakan charger smartphone namun harus memiliki output daya 5V == 1.0A s/d 2.0A. Pastikan kabel yang digunakan adalah micro USB (standar kabel charging smartphone). Kalau belum punya bisa beli di toko komputer atau konter ponsel dengan harga Rp. 35.000,- s/d Rp. 100.000,-an

4. Kartu Memori SD Card & Adapter

SD Card digunakan sebagai ‘memori internal’ Raspi yang menyimpan OS yang akan dijalankan. Spesifikasi yang harus dipenuhi adalah Micro SD Card dengan minimal class 6 (rekomendasi gunakan class 10) dan kapasitas 4 GB. Untuk kapasitasnya tergantung OS yang akan dipilih, namun rekomendasi saya adalah 16 GB. Harga pasaran Mikro SD Card class 6 kapasitas 4 GB adalah mulai dari Rp. 50.000,- s/d Rp. 160.000,-an untuk class 10 kapasitas 16 GB.

Adapter Memory Card (dan Memory Card Reader) Digunakan untuk instalasi OS pada SD Card. Bisa saja memindahkan file instalasi OS ke SD Card dengan cara mengkoneksikan komputer via smartphone (SD Card terpasang di smarthphone), namun hanya berlaku untuk OS NOOBS saja (penjelasan lebih rinci akan dibahas pada postingan selanjutnya), dan kawan akan tetap memerlukan Adapter dan Memory Card Reader jika diharuskan melakukan format ulang pada SD Card (tidak bisa di smartphone). Harga Adapter adalah Rp. 5.000,- dan harga MMC Reader bervariasi dari Rp. 5.000,- s/d Rp. 50.000,-

5. Kabel HDMI atau HDMI to VGA Adapter

Digunakan untuk menghubungkan Raspi dengan monitor. Jika kawan ingin menggunakan TV LCD di rumah (yang support input HDMI) hanya perlu kabel HDMI yang harganya Rp. 50.000,-an saja. Namun jika ingin lebih praktis menggunakan monitor komputer (lebih praktis karena bisa menggunakan keyboard dan mouse di meja komputer sekaligus) bisa menggunakan HDMI to VGA Adapter seharga Rp. 100.000,-. Saran saya? Belilah keduanya, karena kabel HDMI bisa digunakan jika kawan ingin menggunakan Raspi untuk menonton film-film hasil download di TV LCD.

6. Mouse dan Keyboard
Gunakan mouse dan keyboard standar berkabel, bukan yang wireless, untuk digunakan saat instalasi OS. Hal tersebut untuk menghindari masalah tak terdeteksinya mouse dan keyboard saat instalasi OS.

7. Casing

Casing berfungsi sebagai pelindung Raspi Board dari sentuhan, debu, dll. Dalam memilih casing, pertimbangkan juga untuk memilih casing yang kuat (tidak rapuh), tertutup rapat namun tidak kedap terhadap sirkulasi udara, dan mudah dilepas untuk membersihkan Raspi Board secara berkala.

8. Heat sink

Heat sink merupakan aksesoris optional, fungsinya untuk mengurangi panas pada bagian processor. Heat sink biasanya terbuat dari tembaga, kuningan atau aluminium dan harga per-biji cukup murah. Untuk menjaga Raspi Board supaya lebih awet, heat sink sangat saya rekomendasikan.

Demikianlah daftar Perangkat Yang Dibutuhkan Untuk Memulai Menggunakan Raspberry Pi yang harus disiapkan terlebih dahulu. Selamat berkesperimen dengan Raspberry Pi.
